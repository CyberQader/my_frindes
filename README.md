# 👥 Friends List Manager | مدير قائمة الأصدقاء

مشروع بسيط وأنيق بلغة **Python** لتنظيم وعرض أسماء الأصدقاء بطريقة مرتبة ومنظمة رقمياً.

---

### ✨ مميزات الكود (Code Features)
* 🔢 **تنسيق ذكي:** استخدام خاصية `zfill` لضمان ظهور الأرقام بشكل احترافي (01, 02, ...).
* 🔄 **حلقة ذكية:** استخدام `while loop` للمرور على القائمة وطباعتها بدقة.
* 📋 **نظام القوائم:** إدارة مرنة للأسماء داخل `List`.

---
